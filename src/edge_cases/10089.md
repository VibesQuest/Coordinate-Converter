# Fake Legacy Key `10089`

`10089` is not a normal area/zone key.

It is a fake legacy key used for a coordinate authored on the parent world map `UiMapID=947` (`Azeroth`), not on a normal continent or zone map. The known correction corpus point is:

```json
{
  "zoneID": 10089,
  "spawns": {
    "10089": {
      "1": {
        "1": 29.99,
        "2": 89.15
      }
    }
  }
}
```

Source: [tests/data/corrections.json](/home/logon/projects/coordinate-converter/tests/data/corrections.json)

## Why It Is Different

The normal converter expects one of these:

- a regular zone basis
- a direct area basis
- a containing-map identity basis
- an unresolved instance alias

`10089` is none of those. It is effectively:

- parent map: `947`
- source point: `29.99, 89.15`
- desired behavior: figure out which continent slice on `947` contains the point, then translate into that continent's local `0-100` coordinate space

## DBC-Only Calculation

Use `UiMapAssignment` rows for `UiMapID=947`.

For each candidate row:

```text
u = source_x / 100
v = source_y / 100
```

Containment:

```text
UiMin_0 <= u <= UiMax_0
UiMin_1 <= v <= UiMax_1
```

If the point is inside the candidate slice, translate it into the child map:

```text
target_x = (u - UiMin_0) / (UiMax_0 - UiMin_0) * 100
target_y = (v - UiMin_1) / (UiMax_1 - UiMin_1) * 100
```

This is image-rect math using `UiMin/UiMax`. It is not the normal world-space reprojection path.

## Why This Is Not In Runtime Code Right Now

Supporting `10089` cleanly in the converter required introducing a new parent-map alias codepath. That added a lot of structure for one known correction.

Current decision:

- keep the research documented
- keep the inspector tool
- do not keep the generic runtime feature for now
- treat `10089` as explicit debt until there is a second real case or a decision to hardcode per-flavor results

## Tool Usage

Inspect all currently available versions:

```bash
uv run python -m src.edge_cases.inspect_parent_map_alias
```

Emit JSON:

```bash
uv run python -m src.edge_cases.inspect_parent_map_alias --json
```

Inspect a different point:

```bash
uv run python -m src.edge_cases.inspect_parent_map_alias --parent-ui-map-id 947 --x 29.99 --y 89.15
```

When more versions are added to `dbc-output/` and `output/`, the tool will discover them automatically.
